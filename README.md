# QHash
A Simple Gui Hash tool.

QHASH开源绿色哈希计算工具

QHASH简介:

QHASH开源绿色哈希计算工具是本人使用qt写的简洁轻巧，功能强大的哈希计算工具，特色如下：


源代码全部开源，欢迎检查拷贝和修改版本人的源代码，或者自己编译

多平台，支持windows和linux

功能强大，支持常见的MD5,SHA1,SHA256等

绿色，无需安装，无需管理员权限，解压即可运行，对公司和普通用户更加友好

无广告，完全免费

友好清爽的图形用户界面

支持拖放，急速多文件多线程计算

支持保存校验结果到文件

检验比对，高亮匹配的哈希值

更过功能请下载或者编译体验
